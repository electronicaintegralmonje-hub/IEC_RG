This repository contains the source code, data, and numerical simulations supporting **Stage 3 of Internal Elasticity Cosmology (IEC)**, a fully relativistic alternative to ΛCDM where spacetime behaves as a quantum-coherent elastic medium. The model derives linear and higher-order cosmological perturbations from a single scalar field φ coupled to cold matter, resolving σ₈ and H₀ tensions naturally.

## Overview
- **Stage 1**: Background evolution as an elastic alternative to ΛCDM [](https://doi.org/10.5281/zenodo.17559059).
- **Stage 2**: Quantum foundation from coherent vacuum memory in QFT [](https://doi.org/10.5281/zenodo.17560903).
- **Stage 3**: Perturbations up to 3rd order, numerical solutions, and predictions [](https://doi.org/10.5281/zenodo.17740825).

Key predictions: σ₈ = 0.762 ± 0.012, H₀ = 73.8 km/s/Mpc, enhanced ISW, stochastic GW backgrounds.
